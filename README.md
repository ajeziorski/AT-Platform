AT-Platform

We build an interactive platform for people with severe expressive
communication disorders (i.e. the severely speech-language
and writing impaired).

We're working on Ubuntu 12.04 using Python2.7

Contributors (in alphabetic order):
Jerzy Grynczewski, Andrzej Jeziorski, Bartosz Jura, Tomasz Spustek

Dependencies:

* wxPython
* Tkinter
* pyUserInput
* xdotool
* milena (polish speech synthesizer for linux)
